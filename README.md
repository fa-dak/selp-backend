## 📝 소개

사용자의 취향과 상황에 맞춰 맞춤형 선물 꾸러미를 추천하는 서비스 플랫폼입니다.

간단한 설문을 통해 관계, 기념일, 예산, 취향 등을 입력하면 AI 기반 추천 알고리즘을 통해 최적의 선물 꾸러미를 제안합니다.

<br />
